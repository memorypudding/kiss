# KISS Development Roadmap

### Done
- [x] Async engine with parallel module execution
- [x] Self-describing modules with INFO dict (free/paid types, returns)
- [x] AST-based module discovery (no import for --list)
- [x] HIBP module (email, username, phone, hash)
- [x] 9Ghz module (email, username, phone, ip, hash, name, id, ssn, passport)
- [x] Proxy support (HTTP, SOCKS4, SOCKS5) with --proxy and --set-proxy
- [x] API key management (--set-key, env vars, config.json)
- [x] Parser supports: email, username, phone, ip, address, hash, name, id, ssn, passport

### Planned
- [ ] SSID/BSSID lookup via WiGLE and Google API
- [ ] Hash lookup via HashMob API
- [ ] Cross-platform username enumeration (holehe-style)
- [ ] Export functionality (JSON, CSV)
- [ ] Melissa API integration
- [ ] Rate limiting optimization
- [ ] Database caching system
